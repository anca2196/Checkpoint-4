## Getting Started


```bash
  git clone git@github.com:anca2196/Checkpoint-4.git
```

2. Go to the project directory

```bash
  cd checkpoint-4
```

3. Install dependencies

```bash
  npm install
```

4. To start the frontend folder server:

```bash
  npm start
```

5. To start the backend folder server:

```bash
  npm run dev
```

6. Happy coding!
