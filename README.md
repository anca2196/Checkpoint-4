## Getting Started

1. Clone the project

```bash
  git clone git@github.com:anca2196/Checkpoint-4.git
```

2. Go to the project directory

```bash
  cd checkpoint-4
```

3. To start the frontend folder server:

```bash
  cd frontend
  npm install
  npm start
```

4. To start the backend folder server:

```bash
  cd backend
  npm install
  npm run dev
```

5. Happy coding!
